🛒 E-Commerce Website

A modern and responsive E-Commerce Website built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB. This platform provides users with a seamless online shopping experience, allowing them to browse products, manage carts, place orders, and securely interact with the system through an intuitive and attractive interface.

🚀 Project Overview

The E-Commerce Website is designed to simulate a real-world online shopping platform where customers can explore products, add items to their cart, manage orders, and complete purchases efficiently. The project focuses on responsive design, dynamic content management, and user-friendly navigation.

✨ Features
🛍️ Product Catalog Display
🔍 Product Search and Filtering
📦 Product Categories
❤️ Wishlist Functionality
🛒 Shopping Cart Management
➕ Add to Cart Feature
➖ Update Product Quantity
🗑️ Remove Items from Cart
